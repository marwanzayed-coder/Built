# Marwan

Marwan is a JavaScript framework that has a JavaScript function, but with more features

## Installation

Using npm

```bash
npm install marwan --save
```

## Usage

in Javascript File

```javascript
import { marwanPush } from "marwan";

marwanPush([1, 2, 3], 4, 5, 6); // [1, 2, 3, 4, 5, 6]
```
