# Built

Built is a JavaScript framework that has a JavaScript function, but with more features

## Installation

Using npm

```bash
npm install built-function --save
```

## Usage

in Javascript File

```javascript
import { push } from "built-javascript";

let array = [1, 2, 3];
push(array, 4, 5, 6); // [1, 2, 3, 4, 5, 6]
```
